Camera Z up

A Library by Jeremy Behreandt for the Processing programming environment.
Last update, 05/27/2020.

Makes the positive z-axis, (0, 0, 1), the world up axis for 2D and 3D.

Download

Download Camera Z up version 0.0.6 (1) in .zip format.

Installation

Unzip and put the extracted CamZup folder into the libraries folder of your Processing sketches. Reference and examples are included in the CamZup folder.
